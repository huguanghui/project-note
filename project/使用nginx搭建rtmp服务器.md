## 使用

```shell
$ curl http://nginx.org/download/nginx-1.9.9.tar.gz
$ git clone https://github.com/arut/nginx-rtmp-module.git 
$  ./configure --prefix=/usr/local/src/nginx  --add-module=../nginx-rtmp-module  --with-http_ssl_module    
```
