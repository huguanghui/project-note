## 使用

### 标签使用

```shell
#创建标签
ctrl + t
#删除标签
ctrl + w
#切换标签(向前移动)
ctrl + tab
#切换标签(向后移动)
ctrl + shift + tab
```

### 目录控制

```shell
#将选中目录在另一个窗口打开
ctrl ->
ctrl <-
#显示历史文件
alt + 下方向键
#前一次访问的目录
alt + 左方向键
#后一个访问过的目录
alt + 右方向键
```
